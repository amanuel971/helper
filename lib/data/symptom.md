# i,የልብ ድካም
1. የትንፋሽ ማጠር,
2. ድካም እና ድክመት,
3. በእግር፣ በቁርጭምጭሚት ላይ እብጠት,
4. ፈጣን ወይም መደበኛ ያልሆነ የልብ ምት,
5. የማያቋርጥ ሳል ወይም የትንፋሽ ማፍሰሻ,
6. የአካል ብቃት እንቅስቃሴን መቻቻል መቀነስ,
7. ክብደት መጨመር,

# iv,የሚጥል በሽታ

# v,የስኳር በሽታ
1. በተደጋጋሚ የሽንት መሽናት,

5. ድካም እና ድክመት,

8. መደንዘዝ ወይም መንቀጥቀጥ,

# vi,ደም ብዛት
