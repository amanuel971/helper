# 1,የልብ ድካም
የትንፋሽ ማጠር 
ድካም እና ድክመት
በእግር፣ በቁርጭምጭሚት ላይ እብጠት 
ፈጣን ወይም መደበኛ ያልሆነ የልብ ምት
የማያቋርጥ ሳል ወይም የትንፋሽ ማፍሰሻ
የአካል ብቃት እንቅስቃሴን መቻቻል መቀነስ
ክብደት መጨመር
# 2,ደም ግፊት
ራስ ምታት
ማዞር 
የዓይን ብዥታ
የደረት ሕመም
ትንፋሽ ማጠር
ድካም ወይም ድክመት
የአፍንጫ ደም መፍሰስ
# 3,ደም ማነስ
ራስ ምታት
ማዞር ወይም ራስ ምታት
የዓይን ብዥታ
የደረት ሕመም
የትንፋሽ ማጠር
ድካም ወይም ድክመት
የአፍንጫ ደም መፍሰስ
# 4,የሚጥል በሽታ
የንቃተ ህሊና መሳት
የመናድ ችግር
የትኩረት መናድ
የእይታ መዛባት
የንቃተ ህሊና ማጣት
ግራ መጋባት ወይም የማስታወስ ችግር
መኮማተር ና መደንዘዝ
# 5,የስኳር በሽታ
በተደጋጋሚ የሽንት መሽናት
ከመጠን በላይ ጥማት
ያልታወቀ ክብደት መቀነስ
ረሃብ መጨመር
ድካም እና ድክመት
ቁስሎችን ቀስ በቀስ ማዳን
ብዥ ያለ እይታ
መደንዘዝ ወይም መንቀጥቀጥ
ተደጋጋሚ ኢንፌክሽኖች
# 6,ደም ብዛት
ኢንፌክሽኖች
መድማት ወይም መሰባበር
እብጠት ወይም ራስን የመከላከል ሁኔታዎች
የደም መዛባቶች
